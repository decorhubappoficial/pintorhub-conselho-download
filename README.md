# PintorHUB Gestor - Conselho

Canal oficial de instalacao da versao piloto Android do PintorHUB Gestor para o Conselho de Profissionais.

## Baixar

[Baixar a versao mais recente do APK](https://github.com/decorhubappoficial/pintorhub-conselho-download/releases/latest/download/PintorHUB-Gestor-Conselho.apk)

O codigo-fonte do PintorHUB permanece privado. Este repositorio publica somente o instalador assinado.

## Ativacao individual

1. Instale e abra o aplicativo.
2. Preencha sua identificacao e aceite as regras do piloto.
3. Gere a solicitacao `PHREQ1` do aparelho.
4. Envie o codigo em conversa privada para a administracao.
5. Aguarde a licenca individual `PHLIC1`.
6. Importe a licenca no aplicativo.

O APK pode ser compartilhado, mas nao libera o Gestor sem uma licenca assinada para aquele aparelho e aquela instalacao. O gerador de licencas e a chave privada nunca sao distribuidos.
